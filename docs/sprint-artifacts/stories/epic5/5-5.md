## Acceptance Criteria

**AC-1: Slider Adjusts Dashboard Scores**
Given authenticated user with architecture loaded,
When E2E test adjusts a weight slider,
Then dashboard aggregate score changes visibly.

**AC-2: Reset Restores Original Scores**
Given weights adjusted via sliders,
When "Reset Weights" clicked,
Then scores revert to original (unweighted) values.

**AC-3: Export/Import Round-Trip Preserves Weights**
Given weights adjusted via sliders,
When architecture exported and reimported,
Then sliders show same adjusted positions.

**AC-4: v1 Import Defaults Sliders**
Given a v1 YAML file imported,
When architecture loads,
Then all sliders at default positions (1.0).

## Dev Notes

**Slider interaction in Playwright:** shadcn Slider uses Radix `<span role="slider">` with `aria-valuenow`. Most reliable approach: keyboard -- `slider.focus()` then `page.keyboard.press('ArrowLeft')`.

**File download:** Use `page.waitForEvent('download')`, save to temp path, reimport.
